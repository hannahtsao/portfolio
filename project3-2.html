Brainstorming Platform Project

Cloud Financial Holdings - Cloud Technology Internship Projects
Duration: July 2021 - Dec. 2021
Role: Cloud Technology Intern
Skills: Cloud Computing Platform (AWS, GCP), Systems Development (Java Spring Boot), Teamwork

Description

Within the Cloud Technology Division, brainstorming sessions are a vital source of new project ideas. The team wanted to enhance and streamline these sessions through automation.

Objective / Success Metrics

This initiative aimed to create a digital platform that elevates the efficiency of brainstorming. The platform's objective was to automatically generate simplified program architectures based on insights from brainstorming sessions.

Leveraging the department's cloud technology expertise, the platform would be hosted in the cloud, enabling other company departments to utilize it.

Approach

  1. Understanding Current Brainstorming Process:

    Our first step involved comprehensively understanding the existing brainstorming process within the team. We delved into the workflow, identifying pain points and needs.

  2. Platform Development:

    The project proceeded with the development of Application Programming Interfaces (APIs) using Java Spring Boot. These APIs were integrated into the core system. Concurrently, we ensured a seamless fusion of the front-end and back-end components.

  3. Cloud Deployment:

    Upon completion of development, our focus shifted to deployment. Leveraging Amazon Web Services (AWS), we deployed the platform to the cloud. This move facilitated its availability for utilization by various departments within the organization.

Results

The Brainstorming Platform project streamlined brainstorming into 4 steps, enabling structured idea generation and improved teamwork. By successfully automating program architecture creation, the platform reduced errors and saved up to 20% of time. Moreover, by deploying on AWS, the platform became accessible to other departments, encouraging collaboration and knowledge sharing.

In summary, this initiative transformed brainstorming, elevating efficiency and collaboration through automation and cloud technology. The platform was praised by users within the Cloud Technology Division for its user-friendly interface and seamless integration.
